Setup Required

Configure your environment to start using Biddy

Environment Configuration

To get started with Biddy, you need to configure the following environment variables in your Vercel project settings:

Required Variables:

OPENAI_API_KEY =your_openai_api_key_here
DEV_CDP_API_KEY_NAME =your_cdp_api_key_name_here
DEV_CDP_API_KEY_PRIVATE_KEY =your_cdp_api_key_private_key_here
NETWORK_ID =base-sepolia
WALLET_TYPE =coinbase-managed

Development & Production Setup

For initial deployment and testing, we'll use your development CDP credentials in both environments. This allows you to get started quickly. Once everything is working, you can set up separate production credentials if needed.

First Time Setup

On first run after setting these variables, Biddy will create a new CDP wallet and provide you with the wallet data. You'll need to add this data to your environment variables as well.

Important

Make sure to keep your API keys and wallet data secure. Never share them or commit them to version control.

Next Steps:

  1. Go to your Vercel project settings
  2. Navigate to the Environment Variables section
  3. Add each of the required variables above
  4. Deploy your project again
  5. After deployment, Biddy will create a new CDP wallet and provide the wallet data
  6. Add the wallet data as a new environment variable and deploy once more
  7. (Optional) Once everything is working, you can set up separate production credentials
